EAD Ejercicio 1

Documentación electrónica aplicada

Ejercicio en clases luz leds

Para este taller, tuvimos que conectar en nuestros componentes electrónicos, leds y resistencias para poder repasar el como funcionaban. Para esto yo conecte los leds y las resistencias de una manera que fuera un circuito en paralelo, logrando que los 3 leds que termine colocando, se pudieran prender sin riesgos de que se quemen o quitarle vida útil a los leds. Utilicé resistencias de 330ohm y una bateria de 9V para prender leds de color rojo, blanco y verde.

Para conectar la bateria, le corte la terminación para poder conectarlo a la protoboard.
